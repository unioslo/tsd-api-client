## TSD API Client

Python client for TSD API.

## Install

```bash
pip3 install https://github.com/unioslo/tsd-api-client/archive/v2.7.1.zip
```

## tacl

To use the TSD File API, use `tacl`:

```bash
tacl --register
tacl --help
tacl --examples
```

## tsd-s3cmd

To use the TSD s3 API, use `tsd-s3cmd`:

```bash
tacl --register
tsd-s3cmd --guide
tsd-s3cmd --help
```
